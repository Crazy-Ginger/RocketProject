What is NEW at aRocket?

The Collaborative Rocket project page has finally been updated.

We have added an index of NASA's Space Vehicle Design Criteria, SP 8000 and a new ProPEP data file, pepcoded.daf. This page has links to all available Isp calculation software and associated utilities as well as sourcecode ProPEP.

We have acquired three new domains which we will use to promote amateur, experimental and high power rocketry.

We are now Associates of Amazon.com. It would be nice to earn a few dollars to help defray the costs of running aRocket. The Recommended Reading pages are posted.

June 20, 2003. The new domain is up! aRocket now lives at http://www.arocketry.net.

June 18, 2003. The site is beginning to come together. Links have been added to the Site Map, it still needs some work. We have added a page of links to the web sites of aRocket members.

On April 22, 2003 a new renovation of this site began. The Library must be reworked and material that was available will be put somewhere.
